# DevBox Pro - Icon Resources

This directory should contain the following icon files for building the application:

## Required Files

### Windows
- `icon.ico` - Multi-resolution ICO file (16x16, 32x32, 48x48, 64x64, 128x128, 256x256)

### macOS
- `icon.icns` - macOS ICNS icon file with all required resolutions

### Linux
- `16x16.png`
- `32x32.png`
- `48x48.png`
- `64x64.png`
- `128x128.png`
- `256x256.png`
- `512x512.png`

## Creating Icons

1. Start with a 1024x1024 PNG master icon
2. Use tools like:
   - **Windows**: Use `png2ico` or online converter
   - **macOS**: Use `iconutil` command
   - **Linux**: Just resize the PNG to various sizes

## Icon Design Guidelines

- Use a clean, modern design
- Ensure visibility at small sizes
- Use the DevBox Pro brand colors (blue gradient #2563eb to #3b82f6)
- Include transparency where appropriate
